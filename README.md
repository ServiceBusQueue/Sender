# Azure Service Bus Console Application

This project contains a .NET console application that sends messages to an Azure Service Bus queue.

## Prerequisites

- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) installed on your machine.
- An Azure subscription with access to Azure Service Bus.
